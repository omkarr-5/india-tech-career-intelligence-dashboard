# 🚀 India Tech Career Intelligence Dashboard

An end-to-end **Data Analytics Project** that analyzes the Indian technology job market using **Python, Pandas, and Power BI**. The project transforms raw job posting data into interactive dashboards that help users understand hiring trends, salary insights, skill demand, and career opportunities.

---

## 📌 Project Overview

The Indian technology job market is highly competitive and constantly evolving. Thousands of job postings are published every day, making it difficult for students, job seekers, and recruiters to identify meaningful trends.

This project solves that problem by analyzing job market data and presenting key insights through an interactive Power BI dashboard.

---

## ❓ Problem Statement

Raw job listings do not clearly answer important career-related questions such as:

- Which tech roles are most in demand?
- Which skills should I learn?
- Which cities offer the best opportunities?
- Which roles provide the highest salaries?
- Which company types pay better?
- What education is commonly required?

This project converts raw data into actionable insights for better career decision-making.

---

## 🎯 Objectives

- Analyze hiring trends in the Indian tech industry.
- Identify the highest-paying job roles.
- Discover the most in-demand technical skills.
- Compare salaries across cities, company types, and experience levels.
- Build an interactive Power BI dashboard.
- Help users make data-driven career decisions.

---

**Dataset:** India Tech Job Market (2024–2026)

**Dataset Size**

- 5,000 Job Listings

- 109 Unique Skills

- Industry

# 🔍 Key Insights

- Software Engineer is among the most in-demand job roles.
- Python is the most frequently requested technical skill.
- Hybrid and On-Site work models dominate hiring.
- Salary increases significantly with experience.
- Indian Unicorn companies offer higher average salaries.
- Different roles require different combinations of technical skills.
